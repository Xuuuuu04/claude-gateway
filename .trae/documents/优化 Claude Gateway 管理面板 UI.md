# 优化 Claude Gateway 管理面板 UI

## 视觉风格升级：
1. **配色方案**：采用 Anthropic 标志性的米色背景 (#F5F5F0) 和陶土橙 (#AE5630) 作为主色调。
2. **字体设计**：引入 Google Fonts，标题使用衬线体 (Lora)，UI 元素使用无衬线体 (Poppins)，代码使用 JetBrains Mono。
3. **组件重塑**：
   - 将现有的生硬边框卡片改为具有柔和阴影和圆角的“纸张感”容器。
   - 优化按钮样式，增加微交互动画。
   - 重构表格样式，使其更具现代感和可读性。
4. **日志系统优化**：将原本突兀的纯黑日志框改为与品牌色调一致的深色背景。

## 技术调整：
1. 修改 `internal/admin/web/index.html` 中的 CSS 样式。
2. 微调 HTML 结构以支持更现代的布局（如侧边栏或更精致的顶部导航）。
3. 保持现有的 JavaScript 逻辑不变，仅优化渲染生成的 HTML 模板字符串。

## 预期效果：
面板将从一个简单的“工程工具”转变为一个极具 Anthropic 质感的、专业且优雅的管理后台。
